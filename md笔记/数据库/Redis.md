# Redis

## 1、NoSQL

(Not Only SQL)

- 泛指非关系型数据库
- 这些类型的数据存储不需要固定的模式，无需多余的操作就可以横向扩展

**优点**：

- 易扩展
- 大数据量高性能
- 多样灵活的数据模型

**3V和3高：**

- 大数据时代的3V:
  - 海量Volume
  - 多样Variety
  - 实时Velocity
- 互联网需求的三高：
  - 高并发
  - 高可扩
  - 高性能

## 2、Redis概述

高性能键值对数据库

支持的键值数据类型：

- 字符串类型
- 列表类型
- 有序集合类型
- 散列类型（哈希）
- 集合类型
